# Documentation

This directory contains various documentation and reference files for VoteTracker.

## Contents

### Publishing Guides
- `AUR_PUBLISHING_GUIDE.md` - Guide for publishing to AUR (Arch User Repository)
- `AUR_BIN_PUBLISHING_GUIDE.md` - Guide for publishing binary packages to AUR
- `PKGBUILD.aur` - Reference PKGBUILD for AUR source package
- `PKGBUILD.aur-bin` - Reference PKGBUILD for AUR binary package
- `PKGBUILD-axios` - Optional PKGBUILD for axios integration

### Development Documentation
- `IMPLEMENTATION_PLAN.md` - Implementation planning document
- `IMPROVEMENTS.md` - List of improvements and enhancements
- `WINDOWS_OPTIMIZATION.md` - Windows-specific optimizations

### Release Information
- `RELEASE_NOTES_2.7.1.md` - Release notes for version 2.7.1
- `releases/` - Archive of previous package builds
